# my_own_portfolio

my_own_portfolio is a React Hook based progressive web application/portfolio page decided to showcase my background and projects for any devices.

## Tech Stack

- JavaScript
- Node.js
- Express.js
- React Hooks
- Styled-Components
- Webpack
- Babel
- AWS service
- Heroku

## Installation

### dev

1. npm install
2. npm startdev
3. npm run buildDev
4. go to localhost:3000

---

### production
1. npm start
